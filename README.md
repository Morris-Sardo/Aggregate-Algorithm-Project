# Aggregate-Algorithm-Project
The aim of the project is to implement different time siries algorithms on synthetic dataset.

# Setup project
* Import Numpy
* Import matplotlib

# Description
This project has been split in three phases: 

The first phase is to implement a coefficient Beta algorithm for Hidden Markov Model using dynamic programming.

The second part of the project is implement a coefficient Gamma for an Hidden markov Model.

The last part of the project is to implements a n Aggregate Algorithm for calculaitng predicitions output with euqal weights in trhe binary square loss (Brier) game.
